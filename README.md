<p align="center" style="text-align:center">

Tiny set of pure CSS animations that applies to :checked elements.

</p>
<p align="center">
<sub>	... the poer of  <a href="https://github.com/720kb/radiobox.css">radiobox.css</a> + <a href="https://github.com/720kb/checkbox.css">checkbox.css</a> all in one</sub>
</p>

<img src="https://badges.gitter.im/Join%20Chat.svg"/>
</a>
</p>

<p align="center" style="text-align:center">
<b>Installation</b>
</p>
<p align="center" style="text-align:center">
Include the <i>dist/css/checked.min.css</i> in your html page
</p>
<p align="center" style="font-size:12px;">
<sub><i>Include dist/css/yo.min.css instead, if you want to use only the <b>Yo</b> animation, for example.</i></sub>
</p>
